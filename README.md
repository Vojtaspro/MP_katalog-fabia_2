# 🚗 Katalog vozů Škoda Fabia (Maturitní projekt)

Tato interaktivní aplikace slouží jako digitální katalog generací vozu Škoda Fabia. Projekt byl vytvořen v rámci maturitní práce se zaměřením na analýzu a vizualizaci dat v jazyce Python.

## 🌟 Hlavní funkce
* **Filtrování:** Výběr podle generace (I–IV), typu karoserie a motorizace.
* **Technické parametry:** Detailní informace o motoru, výkonu, spotřebě a rozměrech.
* **Dobové ceny:** Přehled cen ve vybraných výbavových stupních.
* **Moderní design:** Tmavé téma s responzivním rozložením (Streamlit framework).

## 🛠️ Použité technologie
* **Python** (backend a logika)
* **Streamlit** (webové uživatelské rozhraní)
* **Pandas** (zpracování dat z Excelu)

## 🚀 Jak spustit projekt
1. Nainstalujte knihovny: `pip install -r requirements.txt`
2. Spusťte aplikaci: `streamlit run app_2.py`
